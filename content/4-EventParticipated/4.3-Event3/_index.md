# "Cloud Mastery 2026 #3 AWS Security & Networking"

### Event Objectives

- Provide an overview of IAM, networking, and system security on AWS.
- Build a foundation for access management, network design, and application protection.
- Connect theory with practice through demos and concrete examples.

### Speaker Details & Presentation Topics

**1. Huynh Hoang Long & Dang Thi Minh Thu - FCAJ Cloud Engineer Ambassador**  
* **Topic:** IAM on AWS  
* **Main content:** Focused on IAM concepts and how to manage identities and access permissions in AWS systems. The session covered IAM best practices, SSO mechanisms, and how to apply SCP and Permission Boundaries for advanced permission control, along with security techniques such as credential rotation and Access Analyzer. The presentation also included live demos.

**2. Lam An Thinh & Nguyen Phan Quoc Viet - Security Engineer Intern**  
* **Topic:** Networking on AWS  
* **Main content:** Presented an overview of AWS networking, including VPC and CIDR design, subnet segmentation, Internet Gateway (IGW) and route table configuration, and the use of VPC Endpoints to optimize internal connectivity.

**3. Lam Tuan Kiet - DevOps**  
* **Topic:** AWS Network & Application Protection  
* **Main content:** Introduced AWS security services such as AWS WAF (Web Application Firewall), AWS Shield, AWS Network Firewall, and AWS Firewall Manager, with a focus on protecting systems and applications from threats.

### Key Highlights

#### 1. IAM and access management
- Understood what IAM is and its role in AWS systems.
- Applied best practices for secure access management.
- Explored SSO mechanisms in enterprise systems.
- Used SCP and Permission Boundaries for advanced permission control.
- Applied credential rotation and Access Analyzer to strengthen security.
- Observed live demos.

#### 2. Networking on AWS
- Understood how networking on AWS works.
- Designed VPC and used CIDR properly.
- Segmented subnets for system design.
- Configured IGW and route tables for traffic routing.
- Used VPC Endpoints to optimize internal connectivity.

#### 3. AWS network and application security
- Learned how AWS WAF protects web applications.
- Understood the role of AWS Shield in DDoS protection.
- Applied AWS Network Firewall for traffic control.
- Used AWS Firewall Manager for centralized security policy management.
