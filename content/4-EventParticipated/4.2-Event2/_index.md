# "Cloud Mastery 2026 #2 DevOps"

### Event Objectives

- Provide an overview of Kubernetes, Infrastructure as Code (IaC), and Elixir.
- Build a foundation for designing, deploying, and operating systems on the Cloud.
- Connect theory with practice through live demos.

### Speaker Details & Presentation Topics

**1. Bao Huynh - Junior Cloud Native Developer - Endava Vietnam**  
* **Topic:** Kubernetes  
* **Main content:** The presentation focused on solving container orchestration challenges with Kubernetes, starting from an overview, architecture, and core components. It then covered key Kubernetes objects, working with manifests, and using kubectl. The session also included live demos, practical guidance on how to start learning Kubernetes, an effective learning roadmap, and supporting tools such as Helm and K9S.

**2. Thinh Nguyen - DevOps Engineer**  
* **Topic:** Infrastructure as Code with Terraform on AWS  
* **Main content:** This session focused on the concept of Infrastructure as Code (IaC) and its role in modern infrastructure management. It included an overview of IaC tools, introductions to AWS CloudFormation, AWS CDK, and Terraform, along with a practical demo of infrastructure deployment on AWS.

**3. Nguyen Ta Minh Triet - SAP Developer Intern, Bosch GSV**  
* **Topic:** Introduction to Elixir  
* **Main content:** Covered core Elixir fundamentals, concurrent programming concepts, and OTP for fault-tolerant systems. The talk also presented how Elixir can be applied in DevOps pipelines, supported by live demos and practical examples.
