# Events I Participated In

During my internship and workshop journey, I joined 3 technical events related to application modernization, DevOps, and AWS security. These events helped me strengthen both technical knowledge and practical system-thinking.

### [Event 1](4.1-Event1/)
&emsp;**Event Name:** GenAI-powered App-DB Modernization workshop

&emsp;**Role:** Attendee

&emsp;**Highlights:**
- Learned modernization strategies for applications and databases.
- Understood Domain-Driven Design (DDD), bounded contexts, and event-driven architecture.
- Compared integration and compute choices (sync/async, VM/containers/serverless).
- Explored Amazon Q Developer and AI support for SDLC activities.

&emsp;**Value Gained:**
- Improved architecture thinking with a business-first mindset.
- Better understanding of phased modernization and ROI-oriented planning.
- Clearer direction for applying event-driven and serverless patterns in real projects.

### [Event 2](4.2-Event2/)
&emsp;**Event Name:** Cloud Mastery 2026 #2 DevOps

&emsp;**Role:** Attendee

&emsp;**Highlights:**
- Learned Kubernetes fundamentals, architecture, and practical deployment workflows.
- Studied Infrastructure as Code (IaC) with Terraform on AWS.
- Explored Elixir basics, concurrency concepts, and OTP for resilient systems.
- Followed live demos showing how these technologies work together in practice.

&emsp;**Value Gained:**
- Built a stronger DevOps foundation across orchestration, automation, and backend thinking.
- Better tool selection mindset based on real requirements.
- Increased confidence in designing scalable and maintainable cloud systems.

### [Event 3](4.3-Event3/)
&emsp;**Event Name:** Cloud Mastery 2026 #3 AWS Security & Networking

&emsp;**Role:** Attendee

&emsp;**Highlights:**
- Learned IAM best practices, SSO concepts, SCP, and permission boundaries.
- Reviewed VPC networking design: CIDR planning, subnets, route tables, IGW, and VPC endpoints.
- Explored AWS protection services: WAF, Shield, Network Firewall, and Firewall Manager.
- Observed practical security and networking demos on AWS.

&emsp;**Value Gained:**
- Better understanding of secure-by-design cloud architecture.
- Improved ability to balance security, performance, and scalability.
- Stronger awareness of access control and network protection in production systems.
